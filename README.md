# Projeto SQL – Análise de Dados em Banco de Restaurante

Este projeto tem como objetivo demonstrar o uso de SQL para consultas, organização e manipulação de dados em um banco de dados relacional de restaurante.

Foram realizadas consultas utilizando filtros por preço, categoria, ingredientes e data de nascimento, além de ordenações, paginação de resultados e criação de tabelas de backup para preservação dos dados. O projeto simula situações reais de análise de informações em sistemas comerciais.

## Funcionalidades exploradas

* Filtros com WHERE
* Ordenação com ORDER BY
* Paginação com LIMIT e OFFSET
* Busca textual com LIKE
* Função YEAR para análise de datas
* Criação de tabela de backup com CREATE TABLE AS
* Organização e extração de dados relevantes

## Tecnologias utilizadas

* MySQL
* SQL
